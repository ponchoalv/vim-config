Instalación
============

Clone the repository:

    $ git clone git://github.com/ponchoalv/vim-config.git ~/.vim
    $ git submodule init
    $ git submodule update

Create a symbolic link of the .vimrc file into your home directory:

    $ ln -s ~/.vim/.vimrc ~/.vimrc
